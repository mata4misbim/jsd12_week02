รูปแบบการเข้าใจ ตอนนี้ยังไม่ค่อยเห็นภาพว่ามันทำงานยังไง อาจจะเป็นเพราะยังไม่ได้ลองต่อ ใช้งานจริงกับตัวเว็บ แต่พอที่จะเข้าใจ Syntax รูปแบบการทำงาน แล้วก็การใช้งาน ตรงนี้ผมไม่ค่อยเข้าใจเรื่องของการใช้งานจริง แต่จะอธิบายในรูปแบบความเข้าใจของผม เหมือนการจดบันทึก

PostgreSQL  เหมือน ตาราง Excel 

ก่อนการใช้งานต้องกำหนด หัวตาราง ไว้ก่อน เช่น ชื่อ อายุ อีเมล

ถ้าแถวไหนไม่มีข้อมูลตามหัวตาราง อาจจะขึ้น Error หรือ Constraint Violation หรืออาจจะเกิดเหตุการแบบชุดข้อมูลไม่ถูกเก็บ ตรงนี้ต้องดูดีๆนิดนึงเพราะเหมือนกับว่า ถ้า ไม่ได้กำหนดตารางหรือหัวข้อ ข้อมูลอาจจะเละและตัว database อาจจะไม่ทำงานต่อ

เรื่อง Syntax ที่ใช้บ่อยๆ SELECT INSERT UPDATE DELETE 

Ex. การใช้ Syntax แล้วก็ วิธีจำการทำงานของ Syntax ที่พอเข้าใจ

# วิธีจำ Syntax ในรูปแบบการดึงข้อมูล SQL PostgreSQL = แปลจากหลังมาหน้า

SELECT name FROM users WHERE id = 1;

# แปล "เลือก (SELECT) ชื่อ จาก (FROM) ตารางผู้ใช้ ที่ (WHERE) ไอดีเท่ากับ 1"

# การเพิ่มข้อมูล

INSERT INTO users (name, age) VALUES ('Oh', 21);

# แปล "ใส่ข้อมูล (INSERT INTO) ลงใน ตาราง users ในช่อง (name, age) ด้วย ค่า (VALUES) ที่เป็น 'Oh' และ 21"

# การแก้ไขข้อมูล หรืออัพเดท

จากอันก่อนที่ข้อมูลของ id 1 = ชื่อคือ Oh ถ้าจะเปลี่ยนให้เป็น 'Metha'

UPDATE users SET name = 'Metha' WHERE id = 1;

# แปล "อัปเดต (UPDATE) ตาราง users โดย ตั้งค่า (SET) ให้ ชื่อเป็น 'Metha' เฉพาะแถวที่ (WHERE) ไอดีเท่ากับ 1" ตรงนี้ต้องเข้มงวดกับ WHERE นิดนึงถ้าลืมอาจจะทำให้ชื่อเปลี่ยนทั้งตาราง

# แล้วก็ การลบข้อมูล

Ex. ยกตัวอย่าง ลบข้อมูล คนที่มี id = 1

DELETE FROM users WHERE id = 1;

# แปล "ลบ (DELETE) ข้อมูลจาก (FROM) ตาราง users เฉพาะแถวที่ (WHERE) ไอดีเท่ากับ 1"
